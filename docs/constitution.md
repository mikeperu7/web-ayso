# Constitución del Proyecto Web AYSO (SSOMA)

## Reglas de Comportamiento para Agentes de IA
1. **Rol Principal:** Eres un asistente técnico operando bajo una arquitectura de múltiples agentes especializados. No asumas roles que no se te han asignado explícitamente en el prompt actual.
2. **Stack Tecnológico Estricto:** - Frontend: Next.js (App Router), React, Tailwind CSS.
   - Lenguaje: TypeScript estricto (no uses `any`).
   - Base de Datos/Auth: Supabase.
   - Componentes: Shadcn UI.
3. **Flujo de Trabajo:** Nunca escribas código funcional sin antes haber validado la arquitectura o el diseño conceptual. 
4. **Claridad sobre Complejidad:** Prioriza código legible y modular. Un archivo debe tener una sola responsabilidad.

# Design System - Web A&SO (SSOMA)

## Colores Principales (Paleta Primaria)
- **Azul Corporativo (Primary):** #006489
- **Verde Ambiente (Secondary):** #438727
- **Naranja Seguridad (Accent/Destacado):** #BB4A05
- **Gris Pizarra (Textos Principales):** #495669

## Colores Secundarios y Fondos (Paleta Secundaria)
- **Oscuro Corporativo (Background Dark/Footer):** #2B3036
- **Negro Puro (Textos muy oscuros):** #000000
- **Blanco Puro (Background Light):** #FFFFFF
- **Detalles Metálicos (Referencial):** Dorado (#B4975A) y Plateado (#8A8D8F) para elementos muy premium.

## Tipografía
- **Fuente Global (Títulos, Párrafos, UI):** `Titillium Web`. (Nota para el desarrollador: Es una Google Font, usar variantes Light, Regular, SemiBold y Bold).
- **Fuente del Logotipo (Solo referencial):** `Ubuntu Bold`.

## Estilo Visual y Componentes
- **Identidad de Marca:** Profesional, limpia y dinámica. El símbolo representa la sinergia de los servicios (Seguridad Ocupacional, Recurso Hídrico y Ambiente) a través de curvas envolventes.
- **Uso de Fondos:** Priorizar fondos blancos o muy claros (`#FFFFFF`) para lectura, usando el Azul Corporativo o el Oscuro Corporativo (`#2B3036`) para secciones de quiebre o el Footer.
- **Contraste de Textos:** En fondos claros, usar el Gris Pizarra (`#495669`) o Negro (`#000000`). NUNCA usar colores primarios brillantes para párrafos largos.
